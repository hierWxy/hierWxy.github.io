Redis

Redisз®Җд»Ӣ

RedisжҳҜдёҖдёӘеҹәдәҺеҶ…еӯҳзҡ„ key-value з»“жһ„ж•°жҚ®еә“гҖӮпјҲдёҺmysqlе…ұеӯҳпјү

В·еҹәдәҺеҶ…еӯҳеӯҳеӮЁпјҢиҜ»еҶҷжҖ§иғҪй«ҳ

В·йҖӮеҗҲеӯҳеӮЁзғӯзӮ№ж•°жҚ®(зғӯзӮ№е•Ҷе“ҒгҖҒиө„и®ҜгҖҒж–°й—»)

В·дјҒдёҡеә”з”Ёе№ҝжіӣ

Redisзҡ„еҗҜеҠЁдёҺжңҚеҠЎ

еҗҜеҠЁ

cmdе‘Ҫд»ӨиЎҢдёӯиҫ“е…Ҙredis-server.exe redis.windows.conf

иҝһжҺҘ

cmdе‘Ҫд»ӨиЎҢдёӯиҫ“е…Ҙredis-cli.exe -h localhost -p 6379 -a

Redisж•°жҚ®зұ»еһӢ

5з§Қеёёз”Ёж•°жҚ®зұ»еһӢд»Ӣз»Қ

RedisеӯҳеӮЁзҡ„жҳҜkey-valueз»“жһ„зҡ„ж•°жҚ®пјҢе…¶дёӯkeyжҳҜеӯ—з¬ҰдёІзұ»еһӢпјҢvalueжңү5з§Қеёёз”Ёзҡ„ж•°жҚ®зұ»еһӢ:

еӯ—з¬ҰдёІ string
е“ҲеёҢ hash
еҲ—иЎЁ list
йӣҶеҗҲ set
жңүеәҸйӣҶеҗҲ sorted set/zset

ж•°жҚ®зұ»еһӢзҡ„зү№зӮ№

Redisеёёз”Ёе‘Ҫд»Ө

еӯ—з¬ҰдёІж“ҚдҪңе‘Ҫд»Ө

SET key value $~~~~~~~$и®ҫзҪ®В·жҢҮе®ҡkeyзҡ„еҖј
GET key $~~~~~~~~~~~~~~~$иҺ·еҸ–жҢҮе®ҡkeyзҡ„еҖј
SETEX key seconds value и®ҫзҪ®:жҢҮе®ҡkeyзҡ„еҖјпјҢе№¶е°Ҷ key зҡ„иҝҮжңҹж—¶й—ҙи®ҫдёә seconds з§’
SETNX key value$~~~~$еҸӘжңүеңЁ key дёҚеӯҳеңЁж—¶и®ҫзҪ® key зҡ„еҖј

е“ҲеёҢж“ҚдҪңе‘Ҫд»Ө

Redis hash жҳҜдёҖдёӘstringзұ»еһӢзҡ„ field е’Ң value зҡ„жҳ е°„иЎЁпјҢhashзү№еҲ«йҖӮеҗҲз”ЁдәҺеӯҳеӮЁеҜ№иұЎпјҢеёёз”Ёе‘Ҫд»Ө:
HSET key field value $~~~~~~~$ е°Ҷе“ҲеёҢиЎЁ key дёӯзҡ„еӯ—ж®ө field зҡ„еҖји®ҫдёә value
HGET key field$~~~~~~~~~~~~~~~~~$иҺ·еҸ–еӯҳеӮЁеңЁе“ҲеёҢиЎЁдёӯжҢҮе®ҡеӯ—ж®өзҡ„еҖј
HDEL key field $~~~~~~~~~~~~~~~~$еҲ йҷӨеӯҳеӮЁеңЁе“ҲеёҢиЎЁдёӯзҡ„жҢҮе®ҡеӯ—ж®ө
HKEYS key $~~~~~~~~~~~~~~~~~~~~~~$иҺ·еҸ–е“ҲеёҢиЎЁдёӯжүҖжңүеӯ—ж®ө
HVALS key $~~~~~~~~~~~~~~~~~~~~~~$иҺ·еҸ–е“ҲеёҢиЎЁдёӯжүҖжңүеҖј

еҲ—иЎЁж“ҚдҪңе‘Ҫд»Ө

RedisеҲ—иЎЁжҳҜз®ҖеҚ•зҡ„еӯ—з¬ҰдёІеҲ—иЎЁпјҢжҢүз…§жҸ’е…ҘйЎәеәҸжҺ’еәҸпјҢеёёз”Ёе‘Ҫд»Ө:
LPUSH key value1 [value2]$~~~~~~~~~~~~~~~~$е°ҶдёҖдёӘжҲ–еӨҡдёӘеҖјжҸ’е…ҘеҲ°еҲ—иЎЁеӨҙйғЁ
LRANGE key start stop$~~~~~~~~~~~~~~~~~~~~~$иҺ·еҸ–еҲ—иЎЁжҢҮе®ҡиҢғеӣҙеҶ…зҡ„е…ғзҙ 
RPOP key$~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~$з§»йҷӨе№¶иҺ·еҸ–еҲ—иЎЁжңҖеҗҺдёҖдёӘе…ғзҙ 
LLEN key$~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~$иҺ·еҸ–еҲ—иЎЁй•ҝеәҰ

йӣҶеҗҲж“ҚдҪңе‘Ҫд»Ө

Redis setжҳҜstringзұ»еһӢзҡ„ж— еәҸйӣҶеҗҲгҖӮйӣҶеҗҲжҲҗе‘ҳжҳҜе”ҜдёҖзҡ„пјҢйӣҶеҗҲдёӯдёҚиғҪеҮәзҺ°йҮҚеӨҚзҡ„ж•°жҚ®пјҢеёёз”Ёе‘Ҫд»Ө:
SADD key member1[member2] $~~~~~~~~~~~~~~~~~~~~~$еҗ‘йӣҶеҗҲж·»еҠ дёҖдёӘжҲ–еӨҡдёӘжҲҗе‘ҳ
SMEMBERS key $~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~$иҝ”еӣһйӣҶеҗҲдёӯзҡ„жүҖжңүжҲҗе‘ҳ
SCARD key $~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~$иҺ·еҸ–йӣҶеҗҲзҡ„жҲҗе‘ҳж•°
SINTER key1 [key2]$~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~$иҝ”еӣһз»ҷе®ҡжүҖжңүйӣҶеҗҲзҡ„дәӨйӣҶ
SUNION key1 [key2]$~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~$иҝ”еӣһжүҖжңүз»ҷе®ҡйӣҶеҗҲзҡ„е№¶йӣҶ
SREM key member1 [member2]$~~~~~~~~~~~~~~~~~~~~~~$еҲ йҷӨйӣҶеҗҲдёӯдёҖдёӘжҲ–еӨҡдёӘжҲҗе‘ҳ

жңүеәҸйӣҶеҗҲж“ҚдҪңе‘Ҫд»Ө

RedisжңүеәҸйӣҶеҗҲжҳҜstringзұ»еһӢе…ғзҙ зҡ„йӣҶеҗҲпјҢдё”дёҚе…Ғи®ёжңүйҮҚеӨҚжҲҗе‘ҳгҖӮжҜҸдёӘе…ғзҙ йғҪдјҡе…іиҒ”дёҖдёӘdoubleзұ»еһӢзҡ„еҲҶж•°гҖӮеёёз”Ёе‘Ҫд»Ө:
ZADD key score1 member1 [score2 member2]$~~~~~~~~~~~~~~~~~~$еҗ‘жңүеәҸйӣҶеҗҲж·»еҠ дёҖдёӘжҲ–еӨҡдёӘжҲҗе‘ҳ
ZRANGE key start stop [WITHSCORES]$~~~~~~~~~~~~$йҖҡиҝҮзҙўеј•еҢәй—ҙиҝ”еӣһжңүеәҸйӣҶеҗҲдёӯжҢҮе®ҡеҢәй—ҙеҶ…зҡ„жҲҗе‘ҳ
ZINCRBY key increment member$~~~~~~~~~~~~~~~~~~~~$жңүеәҸйӣҶеҗҲдёӯеҜ№жҢҮе®ҡжҲҗе‘ҳзҡ„еҲҶж•°еҠ дёҠеўһйҮҸincrement
ZREM key member [member]$~~~~~~~~~~~~~~~~~~~~~~~~~$з§»йҷӨжңүеәҸйӣҶеҗҲдёӯзҡ„дёҖдёӘжҲ–еӨҡдёӘжҲҗе‘ҳ

йҖҡз”Ёе‘Ҫд»Ө

Redisзҡ„йҖҡз”Ёе‘Ҫд»ӨжҳҜдёҚеҲҶж•°жҚ®зұ»еһӢзҡ„пјҢйғҪеҸҜд»ҘдҪҝз”Ёзҡ„е‘Ҫд»Өпјҡ
KEYS pattern$~~~~~~~~~~~~~~~~~~~~~~~~~$жҹҘжүҫжүҖжңүз¬ҰеҗҲз»ҷе®ҡжЁЎејҸ( pattern)зҡ„ key
EXISTS key$~~~~~~~~~~~~~~~~~~~~~~~~~~~~$жЈҖжҹҘз»ҷе®ҡ key жҳҜеҗҰеӯҳеңЁ
TYPE key$~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~$иҝ”еӣһ key жүҖеӮЁеӯҳзҡ„еҖјзҡ„зұ»еһӢ
DEL key$~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~$иҜҘе‘Ҫд»Өз”ЁдәҺеңЁ key еӯҳеңЁжҳҜеҲ йҷӨ key
